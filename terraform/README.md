# Terraform

    - EC2 Instance Deploy

## Usage
    - site.tf: script with the site configurations
    - variables.tf: environment variables
    - vpc.tf: VPC configuration

# Requirements

#### User host
    - Terraform 0.11+


## Provide AWS EC2 Instance
For an new EC2 Instance

    - set AWS environment variables.tf

`
    - $ terraform init
`

`
    - $ terraform plan 
`

`
    - $ terraform apply
